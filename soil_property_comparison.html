comparing soil property data

Aim

This document compares gridded soil data to the data gathered at the EPs.

For this, I’m using the dataset ID 31210 from the Soil abiotic (core) project’s Soil Sampling Campaign 21 from BExis Soil sampling campaign 2021, all experimental plots (EP), 0-10 cm - carbon and nitrogen concentrations with the following variables:

[1] "Total_C"     "Inorganic_C" "Organic_C"   "Total_N"     "CN_ratio"

There is also a pH dataset from the abiotic soil core team with the dataset ID 31074 Soil sampling campaign 2021, all experimental plots (EP), 0-10 cm - soil pH. “The soil pH was measured in a weak (0.01 M) calcium chloride solution using a pH meter.”
